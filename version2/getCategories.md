**getCategories**
----
  Returns the Event Categories in JSON format.

* **Version:**

  2

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**
 
   `include_label [bool]`
   
   **Optional:**
 
   none

   **Conditional:**

   none

* **Success Response:**
    
    `include_label:true`

    ```javascript
    categories: [
		{
			category: 10,
			label: "Sporting"
		},
		{
			category: 11,
			label: "Academic"
		},
		{
			category: 12,
			label: "Staff Events"
		},
		{
			category: 13,
			label: "Art Drama and Music"
		},
		{
			category: 14,
			label: "Boarders Events"
		},
		{
			category: 15,
			label: "Excursions"
		}
    ]
  ```

  `include_label:false`

    ```javascript
    categories: [
		10,
		11,
		12,
		13,
		14,
		15
	]
  ```
 
* **Error Response:**

   Required `[field_name]` not supplied
    ```javascript
    __invalid: {
      [field_name]: "field is required."
    }
    ```
    
    Bool `[field_name]` not a valid date
    ```javascript
    __invalid: {
      [field_name]: "Value is not a valid bool."
    }
    ```
    
* **Sample Parameters:**

  ```javascript
    { "include_label" : true }
  ```
