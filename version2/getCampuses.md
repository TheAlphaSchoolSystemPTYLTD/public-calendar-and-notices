**getCampuses**
----
  Returns the School Campuses in JSON format.

* **Version:**

  2

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**
 
   `include_label [bool]`
   
   **Optional:**
 
   none

   **Conditional:**

   none

* **Success Response:**
    
    `include_label:true`

    ```javascript
    campuses: [
        {
            campus: "ARG",
            label: "Argyle Street Campus"
        },
        {
            campus: "JU",
            label: "Junior School (Billabong Road)"
        },
        {
            campus: "SE",
            label: "Senior School (Curlew St)"
        },
        {
            campus: "THO",
            label: "Thompson Street Campus"
        }
    ]
  ```

  `include_label:false`

    ```javascript
    campuses: [
        "ARG",
        "JU",
        "SE",
        "THO"
    ]
  ```
 
* **Error Response:**

    Required `[field_name]` not supplied
    ```javascript
    __invalid: {
      [field_name]: "field is required."
    }
    ```
    
    Bool `[field_name]` not a valid date
    ```javascript
    __invalid: {
      [field_name]: "Value is not a valid bool."
    }
    ```
    
* **Sample Parameters:**

  ```javascript
    { "include_label" : true }
  ```

* **Sample GET:** (With URL Encoded `token`)

  ```HTML

  ```
